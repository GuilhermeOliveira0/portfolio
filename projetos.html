API de Vendas

C# .NET PostgreSQL API REST Entity Framework

Sistema completo de gerenciamento de vendas desenvolvido com arquitetura RESTful, implementando as melhores práticas de desenvolvimento back-end.

Funcionalidades Principais:

  • CRUD completo de produtos, clientes e pedidos
  • Autenticação e autorização com JWT
  • Validação de dados com FluentValidation
  • Documentação automática com Swagger
  • Integração com banco de dados PostgreSQL
  • Testes unitários e de integração
  • Arquitetura em camadas (Repository Pattern)
